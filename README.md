# 🎮 Number Guessing Game (Java)

## 📌 Project Overview

The **Number Guessing Game** is a simple Java console-based game where the player has to guess a randomly generated number between **1 and 100**.

The player has a **limited number of attempts** to guess the correct number. After each guess, the program provides hints whether the number is **too high** or **too low**.

This project demonstrates the use of **Java basics**, including loops, conditional statements, and random number generation.

---

## ✨ Features

* 🎲 Random number generation between **1 and 100**
* 🔢 Maximum **10 attempts** to guess the number
* ⬆️ Hint if the guess is **too high**
* ⬇️ Hint if the guess is **too low**
* ✅ Displays success message when guessed correctly
* ❌ Shows correct number if attempts are finished

## 🎮 Game Rules

1. The system generates a **random number between 1 and 100**.
2. The player must guess the number.
3. The player has **10 attempts**.
4. The program gives hints:

   * **Too High**
   * **Too Low**
5. If the player guesses correctly, the game ends.
